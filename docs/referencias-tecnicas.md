# Referências técnicas verificadas em julho de 2026

- Next.js App Router e instalação oficial.
- Tailwind CSS para Next.js com PostCSS.
- Better Auth com Next.js e adapter oficial Drizzle.
- Drizzle ORM com PostgreSQL.
- Playwright para testes E2E.
- Vitest para testes unitários.

As versões usadas foram consultadas no registro npm durante a geração. Antes de atualizar dependências, executar `npm run check` e testar em staging.
