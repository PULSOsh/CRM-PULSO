# Fluxos principais

## Venda completa

```text
Lead
→ primeiro contato
→ qualificação
→ briefing
→ diagnóstico
→ proposta
→ negociação
→ aceite
→ contrato
→ assinatura
→ recebível
→ confirmação de pagamento
→ projeto
→ aprovação
→ entrega
→ garantia
→ suporte/manutenção
```

## Briefing

1. Criar a partir da oportunidade e produto.
2. Gerar página segura.
3. Enviar por e-mail ou copiar link.
4. Cliente inicia e o sistema salva rascunho.
5. Cliente conclui.
6. Oportunidade avança.
7. Tarefa de análise é criada.
8. Respostas alimentam proposta.

## Proposta

1. Criar de briefing concluído ou justificativa de pulo.
2. Montar blocos.
3. Adicionar itens, adicionais e condições.
4. Validar margem.
5. Publicar snapshot.
6. Enviar.
7. Registrar visualizações.
8. Cliente aceita, rejeita ou pede alteração.
9. Aceite congela versão.
10. Contrato é preparado em rascunho.

## Condição alternativa

1. Cliente informa entrada, parcelas, datas e comentário.
2. Sistema não altera a proposta.
3. Administrador recebe alerta.
4. Administrador aprova, rejeita ou contrapropõe.
5. Nova versão é gerada.
6. Administrador revisa e publica.

## Contrato

1. Gerar rascunho da proposta aceita.
2. Revisar cláusulas e signatários.
3. Escolher assinatura interna, ZapSign ou upload.
4. Enviar após confirmação.
5. Receber eventos.
6. Armazenar evidências e PDF final.
7. Preparar recebíveis.

## Cobrança

1. Contrato assinado prepara parcelas.
2. Administrador revisa.
3. No modo manual, registra dados de PIX, cartão externo ou dinheiro.
4. No gateway futuro, clica em gerar cobrança.
5. Webhook confirma pagamento.
6. Sistema gera recibo.
7. Projeto pode ser iniciado após regra configurada.

## Projeto

1. Criar a partir de template do produto.
2. Ajustar etapas sem alterar template.
3. Solicitar materiais.
4. Executar tarefas e registrar horas.
5. Enviar entregável para aprovação.
6. Controlar revisões.
7. Obter aprovação final.
8. Concluir e gerar entrega.
9. Iniciar garantia.
10. Ofertar manutenção.

## Suporte extra

1. Cliente abre ticket.
2. Triagem define cobertura.
3. Se fora do escopo, criar oportunidade vinculada.
4. Seguir proposta e aceite.
5. Executar trabalho.
6. Registrar horas, custo e satisfação.

## Exclusão

1. Mostrar dependências e impacto.
2. Pedir confirmação.
3. Mover entidade recuperável para lixeira.
4. Permitir restauração por 30 dias.
5. Documentos críticos são arquivados, cancelados ou estornados.
6. LGPD pode anonimizar sem apagar evidências obrigatórias.
